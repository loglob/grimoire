Search Options

A query consists of search terms linked with the operators ;, | and ,.
; and , are conjunctions, | is a disjunction. ; binds the weakest and , the strongest.

Search terms consist of the following:

For all Games

  • prepared, which matches if a spell is currently prepared (only on spell list page)
  • $ followed by any part of a spell's material components, which matches if a spell's material components contain that string.
  • A verbatim casting time value, such as 1 bonus action or 3 [action points].
  • Any part of a spell's name. Any spell which has that string contained in its name is matched.
  • A spell's full name, prefixed by \. For example, \shield matches Shield but not Shield of Faith.
  • Any part of a spell's description, prefixed by /. Prefixing with // matches a single word verbatim instead. Words may be separated by space, or any non-letter characters, both matches are attempted.
  • Any of these terms, prefixed with !. Matches only spells which aren't matched by the following search term.

For D&D 5e

  • ritual, which matches if a spell is a ritual
  • concentration, which matches if a spell requires concentration
  • upcast, which matches if a spell has an upcast
  • verbal, which matches if a spell requires verbal components
  • somatic, which matches if a spell requires somatic components
  • material, which matches if a spell requires material components
  • $$, which matches if a spell requires expensive material components
  • L which matches the spell's level against a value or range. For example, L3 matches 3rd level spells, and L6-9 spells of 6th through 9th level.
  • : followed by a source's verbatim shorthand, which matches every spell from that source.
  • # followed by any part of a spell's source hint, which matches if a spell's source hint contains that string.
  • A verbatim duration value, such as 1 minute.
  • A spell school, such as enchantment.
  • A caster class, such as wizard.

For Goedendag

  • A verbatim arcanum, such as nature
  • A verbatim power level, such as petty
  • A verbatim distance value, such as touch
  • combat, which matches if a spell has the combat (C) tag
  • reaction, which matches if a spell has the reaction (R) tag

For Pathfinder

  • A verbatim trait, such as concentrate
  • A verbatim tradition, such as arcane
  • A verbatim range, such as 30 feet
  • A verbatim area, such as 5-foot burst
  • A verbatim duration, such as 1 minute
  • A verbatim casting time, such as single action
  • A verbatim saving throw, such as will
  • $, followed by a component, such as verbal
  • The L operator, exactly like 5e.
Even if one of these cases match, all others are tried anyways. In particular, any term used is tested against the spells' names. The same applies to how ! search terms are matched.
Text matches are case-insensitive and any whitespace is trimmed and squeezed.
